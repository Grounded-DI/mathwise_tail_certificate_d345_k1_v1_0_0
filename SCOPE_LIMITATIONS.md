# Scope Limitations

This package proves only the fixed exact-conductor theorem

\[
C(\{3,4,5\},1)=80.
\]

It does not claim:

1. a general BEGL theorem;
2. results for any \(D\ne\{3,4,5\}\);
3. results for \(k\ne1\);
4. a density theorem;
5. an asymptotic theorem;
6. a classification theorem;
7. independent third-party acceptance;
8. novelty beyond the fixed-case certificate unless externally verified.

The certificate is replayable from the included files.
